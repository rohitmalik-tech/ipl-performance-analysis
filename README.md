# 📊 IPL Performance Analysis Dashboard

## 📌 Overview

This project is an interactive **IPL Performance Dashboard** built using Power BI.
It provides insights into team performance, player statistics, match results, and trends across IPL seasons.

---

## 🚀 Features

* 📈 Team performance analysis
* 🏏 Player statistics (runs, wickets, etc.)
* 📊 Match results & win trends
* 🔍 Interactive filters for deep analysis
* 📅 Season-wise insights

---

## 🛠️ Tools & Technologies

* Power BI (.pbix file)
* Data Visualization
* Data Analysis
